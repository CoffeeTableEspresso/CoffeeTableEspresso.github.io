About Me

I'm a software developer living in Vancouver. I can be found on GitHub as CoffeeTableEspresso. I'm a Combined Honours Mathematics & Computer Science student at the University of British Columbia, with an interest in programming language theory and compilers.

About This Site

I made this website with raw HTML and CSS as a protest against some of the bloated websites we see online today. I tried to make it look vaguely like the standard Ubuntu terminal. I used the C Preprocessor to template this site, in order to make sure no-one ever hires me for a webdev role. (Linkedin version: I used a HtML teMpLAtinG tOoL tO cReAtE mY sTaTic WeBsITe.)
