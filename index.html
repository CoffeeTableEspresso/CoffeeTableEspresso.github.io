About Me

I'm a software developer living in Vancouver. I can be found on GitHub as CoffeeTableEspresso. I'm a Combined Honours Mathematics & Computer Science student at the University of British Columbia, with an interest in programming language theory and compilers.

About This Site

I made this website with raw HTML and CSS as a protest against some of the bloated websites we see online today. I tried to make it look vaguely like the standard Ubuntu terminal. I used the C Preprocessor to template this site, in order to make sure no-one ever hires me for a webdev role. (Linkedin version: I used an HtML teMpLAtinG tOoL tO cReAtE mY sTaTic WeBsITe.)

Stuff I Like

  • vim — Scriptable text editor.
  • Azure Pipelines — Continuous integration with support for Windows, Linux, and MacOS. I use Azure Pipelines to test out YASL.
  • JetBrains IDEs — My go-to IDEs. I use these whenever they have support for a language I'm using ( which is most languages I usually use).

Projects

I work on many projects, but my main focus currently is YASL, a small embeddable scripting language I've built from scratch in pure C99.

I've also managed to compile the YASL interpreter to WASM. The interpreter can be found online here, if you'd like to try YASL without downloading anything.
