About Me

I'm a software developer living in Vancouver. I can be found on GitHub as CoffeeTableEspresso. I'm a Combined Honours Mathematics & Computer Science student at the University of British Columbia, with an interest in programming language theory and compilers.

About This Site

My website's kind of fun for me. I get to do drawings on that. It's kind of fun.
— Jeff Bridges

I made this website with raw HTML and CSS as a protest against some of the bloated websites we see online today. I tried to make it look vaguely like the standard Ubuntu terminal. I used the C Preprocessor to template this site, in order to make sure no-one ever hires me for a webdev role. (Linkedin version: I used an HtML teMpLAtinG tOoL tO cReAtE mY sTaTic WeBsITe.) Also, yes, those are the forbidden <marquee> and <blink> tags at the top of the page. The <marquee> tag is merely deprecated, but I had to animate the <blink> tag with CSS myself, since modern web browsers have plunged us into a <blink>-less dark age.
