About Me

I'm a software developer living in Vancouver. I can be found on GitHub as CoffeeTableEspresso. I'm a Combined Honours Mathematics & Computer Science student at the University of British Columbia, with an interest in programming language theory and compilers.

I made this website with raw HTML and CSS as a protest against some of the bloated websites we see online today. I tried to make it look vaguely like the standard Ubuntu terminal. Some other programs and tools I like to use:

  • vim — scriptable text editor
  • Azure Pipelines — continuous integration for GitHub projects. Easy to use with support for Windows, Linux, and MacOS.
  • JetBrains IDEs — my favourite IDEs

My Thoughts on Programming languages

Anyone who talks with me knows I criticise almost every programming language I have ever used. Here's my thoughts on some languages. Don't @me.

  • C — One of my favourite languages. Portable and low-level.
  • C++ — Not very well designed but gets the job done when you need need some abstraction on top of C.
  • Python — Good general purpose language for scripts and prototypes.
  • Julia — For anything numeric.
  • Bash — For portable shell scripting. (On most platforms at least.)
  • Java — Like C++ but with all the useful parts chopped off.
  • JavaScript — Two words: Stockholm Syndrome.
  • TypeScript — Doesn't really fix JavaScript, but does make it much more bearable.
  • Lisp — You forget about the parentheses surprisingly quickly.
  • OCaml — My favourite functinal language. I'd love a chance to use this on a larger project sometime.
  • Ruby — Nothing against Ruby, I just don't see a compelling reason to every use it over Python.
  • MATLAB — Please use Julia instead, thanks. Or even Python.
  • R — See my thoughts on MATLAB.
  • Go — The concurrency is just about the only decent part of the language.
