about

YASL (Yet Another Scripting Language) is a language i'm currently working on. i'm writing a bytecode interpreter for YASL at the moment, which can be found here.

types

currently, for numeric types, YASL supports ints (64-bit) and floats (64-bit), and the usual adding, subtracting, multiplying, dividing (dividing always returns a float, as that's what you want in almost all cases), negation of the two types, with the expected conversion of ints to floats when using both ints and floats in an expression.

YASL also supports bools, (true and false), as well as a nil value. negation of bools can be done with the usual ! notation. YASL also supports and and or, for all types, with false and nil being falsey, and everything else being truthy. and and or shortcircuit, and will return either their first or second operand.

control flow

YASL currently has the ternary operator, if-statements, and while-loops for control flow.

sample if-statement:

var x = true
var y = false
if x and y {
 print 11
} elseif x {
 print 10
} elseif y {
 print 1
} else {
 print 0
}

syntax

YASL uses the "usual" infix notation for operators.

variables are declared with (e.g.) var x = 5. variables are assigned with (e.g.) x = 6.

values can be printed using print value.
