1 – Introduction

YASL is a programming langugage.

2 – Basic Concepts

This section describes the basic concepts of the language.

2.1 – Values & Types

YASL is dynamically typed. There are no type definitions. All values carry their own type.

The basic types in YASL are: nil, boolean, int64, and float64. The type nil has a single value nil, whose main use is being different from every other value. The type boolean has two values, true and false. int64 represents integers to 64-bits of precision. float64 represents floating-point values using 64-bits of precision.

3 – The Language

This section describes the lexis, syntax, and semantics of YASL. Language constructs will be explained using the usual EBNF notation.

3.1 – Lexical Conventions

Names in YASL consist of a numbers, letters, and underscores.

the following keywords are reserved and cannot be used as names: 

     and    class   defun   else    elseif    false   
     if     nil     or      return  true      while

3.2 – Variables

Variables store values. There are two kinds of varialbes in YASL: local and global.

All variables are global unless not declared in the global scope, in which case the variable is local.

3.3 – Statements

YASL has if- and while- statements with the following syntax: stmt = "while" expr "{" {stmt} "}" stmt = "if" expr "{" {stmt} "}" {"elseif" expr "{" {stmt} "}"} ["else" "{" {stmt} "}"]

Both false and nil are considered falsey, and all other values are truthy. In particular, 0 and 0.0 are truthy.

3.4 – Expressions

3.4.1 – Arithmetic Operators

YASL supports the following arithmetic operators:

  • +: addition
  • -: subtraction
  • *: multiplication
  • /: float division
  • -: unary minus

all operators, with the exception of float division, which always returns a float, work as follows: If both operands are integers, the result is an integer. Otherwise, the operands are converted to floats and the result is a float.

3.4.2 – Logical Operators

The logical operators in YASL are !, and, and or. Like the control structures, the logical operators consider false and nil as falsey, and anything else as truthy.

! always returns a boolean. and returns its first argument if that argument is falsey, otherwise it returns its second argument. or returns its first argument if that argument is truthy, otherwise it returns its second argument.
