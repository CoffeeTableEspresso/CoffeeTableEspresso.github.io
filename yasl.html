2 – Basic Concepts

This section describes the basic concepts of the language.

2.1 – Values & Types

YASL is dynamically typed. There are no type definitions. All values carry their own type.

The basic types in YASL are: nil, boolean, int64, and float64. The type nil has a a single value nil, whose many use is being different from every other value. The type boolean has two values, true and false. int64 represents integers to 64-bits of precision. float64 represents floating-point values using 64-bits of precision.

3.4.1 – Arithmetic Operators

YASL supports the following arithmetic operators:

  • +: addition
  • -: subtraction
  • *: multiplication
  • /: float division
  • -: unary minus

all operators, with the exception of float division, which always returns a float, work as follows: If both operands are integers, the result is an integer. Otherwise, the operands are converted to floats and the result is a float.
